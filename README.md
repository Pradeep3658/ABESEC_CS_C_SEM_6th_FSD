# ABESEC_CS_C_SEM_6th_FSD

## Project Overview
This repository contains the Full Stack Development (FSD) projects and exercises for the 6th semester at ABESEC. It includes various day-wise tasks and assignments to help students learn and practice FSD concepts.

## Folder Structure
- **day_01**: Contains exercises related to promises in JavaScript.
- **day_02**: File reading and writing examples.
- **day_03**: Asynchronous file operations and directory handling.
- **day_04**: Scripts for working with OS and file statistics.
- **day_5**: File utilities and services for reading and writing JSON files.
- **day_6**: Scripts for deleting and updating JSON data.

## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/Pradeep3658/ABESEC_CS_C_SEM_6th_FSD.git
   ```
2. Navigate to the desired folder and run the scripts using Node.js.
   ```bash
   cd day_6
   node delete.js
   ```

## Prerequisites
- Node.js v22.14.0 or higher
- Basic knowledge of JavaScript and Node.js
